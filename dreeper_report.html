A detailed report on the Dreeper Origin

TLDR; I can't accept this Origin into the server yet. Read below for reasons.

  • pack.mcmeta is incomplete. A description is not provided.
    • Line 4: "description": "Your Description Here!" It is highly recommended to add a description to datapacks, despite them not normally showing up in a server menu.
  • Some abilities have issues, whether those be technical or other reasons.
    • 3 abilities do not have a clear reason for their presence (origins:slow_falling, drep:speedy, drep:small_boy)
    • 3 abilities have technical issues
      • 1 is hidden from the ability list (drep:lush_colour)
      • 1 has a file typo (origin:no_shield)
      • 1 has multiplier issues (drep:explosive)
  • No malcious code was found within the datapack.

= Suck it Green Boi (Skin Color Change) drep:lush_colour Source: Original?

Within this file, the "hidden" value is set to true. Despite this having a name and description, the "hidden" value hides it from the Origin ability list in-game. It is recommended that you set this to false, or remove line 3 entirely. I must note that if you want the exact colors from our current Creeper origin, you need 0.7 red instead of 0.6 red.

- Deathly Allergic (Damage Near Cats) drep:deathly_allergy Source: OriginsPlus (/src/main/resources/data/originsplus/powers/scared_of_cats.json)

- Explosive (Fire Vulnerability) drep:explosive Source: OriginsPlus (/src/main/resources/data/originsplus/powers/flammable.json)

The "Explosive" ability causes the player to take more damage while in fire. Despite this being the intended purpose, the value by which the damage is multiplied is set to 1. This means that there is no damage being multiplied, making the drawback ineffective. You must set this to a higher value in order to get your origin approved.
